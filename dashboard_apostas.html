⚙️ Como configurar o Firebase

1. Acesse console.firebase.google.com
2. Clique em Criar projeto → dê um nome (ex: gabriel-metodos) → desative Google Analytics → Criar
3. No menu lateral clique em Build → Realtime Database → Criar banco de dados → escolha a região → Iniciar no modo de teste
4. No menu lateral clique em ⚙️ Configurações do projeto → Seus apps → </> Web → registre o app → copie o objeto firebaseConfig
5. Abra o arquivo dashboard_apostas.html num editor de texto (Notepad) → procure por COLE_SUA_API_KEY → substitua com os valores copiados
6. Salve e abra o arquivo no Chrome — pronto! 🎉
